======================================================
  PUMMEL PARTY MOBILE GAME — Unity 2022 LTS
======================================================

QUICK START (3 steps):
  1. Open the UnityMobileGame folder in Unity Hub
  2. In Unity: Tools > Setup Game Scene
  3. Press Play ▶

FULL INSTRUCTIONS:
  Open SETUP_GUIDE.html in your browser.

SCRIPTS:
  Assets/Scripts/PlayerController.cs   — Rigidbody movement + joystick
  Assets/Scripts/CameraFollow.cs       — Smooth 3rd-person camera
  Assets/Scripts/KnockbackObject.cs    — Physics knockback on cubes
  Assets/Scripts/FloatingJoystick.cs   — Mobile virtual joystick
  Assets/Scripts/SceneSetup.cs         — Editor utility (builds scene)

UNITY VERSION: 2022.3 LTS (any patch)
RENDER PIPELINE: Built-in (Standard)
TARGET PLATFORM: Android / iOS / PC
